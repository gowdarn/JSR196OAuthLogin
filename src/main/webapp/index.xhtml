JSR196 Multilogin Study

This project aims to implement a reusable implementation of a JSR196 authenticators

Introduction

This project implements native and social (Twitter, Google+, Facebook and others) login, as well as an OAuth provider using JSR196 as much as possible. In certain cases JSF is also used, for example to show the OAuth consent screen, the Login page and the Registration page. You're free to customize these pages as you see fit.
